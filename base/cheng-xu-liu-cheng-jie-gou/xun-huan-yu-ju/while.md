# while

```cpp
while(循环条件){
    循环体
}
//只要循环条件结果为真，就执行循环语句
//https://raw.githubusercontent.com/ermaozi/get_subscribe/main/subscribe/clash.yml
```

```cpp
int num = 0;

while(num<10){
    cout << num << endl;
    num++;
}
//while(true) 则会进入死循环

```
