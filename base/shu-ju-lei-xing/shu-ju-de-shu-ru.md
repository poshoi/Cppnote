# 数据的输入

看一下不同类型的数据的输入

```cpp
#include<iostream>
using namespace std;

int main(){
    // 1. 整形 
    int a = 0;
    cout << "请输入" << endl;
    cin >> a;
    cout << "a 是多少？" << a << endl;
    
    //2. 浮点型
    float f = 3.14f;
    cout << "请输入" << endl;
    cin >> f;
    cout<< "浮点型 F 为"  << endl;
}
```
